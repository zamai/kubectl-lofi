# Installation

1

Install krew

The kubectl plugin manager. If you don't have it yet.

$ brew install krew
2

Install the plugin

One command. No YAML files to write. For once.

$ kubectl krew install \
--manifest-url=https://raw.githubusercontent.com/zamai/kubectl-lofi/main/lofi.yaml
3

Vibe

Put on your headphones. You've earned this.

$ kubectl lofi

# FAQ

Q: Does this actually improve my Kubernetes skills?
Legally, we cannot confirm or deny this. But also yes.
Q: Can I use this in production?
This is production. The lo-fi never stops in production.
Q: Why macOS only?
Because Mac users need the most emotional support while doing DevOps.
Q: My cluster is on fire. Lo-fi first or fix it first?
Lo-fi first. Always lo-fi first. You can't fix anything in a state of panic.
Q: Does it work with OpenShift?
It works with anything that makes you stressed, so yes, especially OpenShift.
Q: How does it work?
It streams MP3 audio. In your terminal. While you kubectl. There is no AI, no blockchain, no machine learning. Just vibes.
